How To Use Board Rollup Sync

Board Rollup Sync is a monday.com board view. It manually copies selected items from multiple source boards into one target rollup board for dashboard reporting.

Install

  1. Install the app from the monday.com app marketplace or install link.
  2. Open the target rollup board in monday.com.
  3. Add a new board view and choose Board Rollup Sync.

Configure

  1. Enter source board IDs separated by commas.
  2. Enter the target rollup board ID and target group ID.
  3. Enter target column IDs for source board, source item link, and sync hash tracking.
  4. Map source column titles to target column IDs, one mapping per line.

Preview And Sync

  1. Click Preview sync to see planned creates, updates, and skips.
  2. Review the plan before writing to the target board.
  3. Click Run manual sync to create or update rollup items.

Permissions

The app requests board read access to inspect source boards and board write access to create or update items on the configured target rollup board. It does not edit the configured source boards.
