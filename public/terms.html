Use of the App

Board Rollup Sync provides manual rollup sync tools for monday.com boards. Users are responsible for configuring source board IDs, target board IDs, target group IDs, and column mappings correctly.

The app works inside monday.com and requires a valid monday.com account with permission to install and use board views. Use of monday.com remains subject to monday.com's own terms and policies.

Board Data Changes

Board Rollup Sync writes to monday.com only when a user runs a manual sync. The app creates or updates items on the configured target rollup board and does not edit the configured source boards.
