Privacy Policy

Last updated: June 3, 2026

Board Rollup Sync is a monday.com board view operated by Radom UG. The app helps teams copy selected source board items into one configured rollup board for reporting and dashboard workflows.

Data We Access

When installed, the app can read monday.com board IDs, board names, group IDs, item IDs, item names, column titles, column types, and column values needed to build a rollup preview and prepare sync operations.

How Data Is Used

Board data is used only to display the preview, calculate create/update/skip operations, and perform a manual sync when a user clicks Run manual sync. The app writes only to the configured target rollup board.

Storage

The app stores user configuration and source-to-rollup item mappings in monday.com instance storage. Board item data is not stored in an external database by Board Rollup Sync.

Third Parties

The app is hosted on Vercel and uses monday.com's API and SDK to operate inside monday.com. The app does not use advertising trackers and does not sell personal data.

Retention

Configuration and sync mappings remain available while the app is installed and monday.com instance storage keeps the data. Users can edit configuration or remove the app view from monday.com.

Security

Board Rollup Sync requests only the monday.com OAuth scopes required for the board view. At launch, the app requests board read access and board write access so it can create and update rollup items on the user's behalf.

Contact

For privacy questions, support, or deletion requests, contact support@konverter-pro.de.
